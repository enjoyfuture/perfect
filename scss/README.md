## 该文件夹下用来定义 sass 样式

目前结构模式如下:

* perfect.scss 主文件
* core 基础样式库
 - components 定义各种基础样式库
 - mixins 定义样式函数库
 - utils 工具类样式库
 - variables 定义变量
 - mixins.scss 函数库入口样式
 - normalize.scss 引入标准化样式库,[参加官方](https://github.com/necolas/normalize.css)
 - reset.scss 定制重置样式
 - utils.scss 工具类样式库
 - variables.scss 定义样式变量
